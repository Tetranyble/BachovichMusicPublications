Admin Home

Welcome to the Bachovich Music Publications Admin Dashboard! From here you can enter, update and delete the artists and products displayed on bachovich.com.

Helpful Hints:

Entering new products and artists

All sheet music, media, and rentals must be associated with a BMP artist. When entering new products, be sure to make a note of the artist id. If the product is composed by a new BMP artist, enter the artist info into the Artists table before entering the product and make a note of the artist id.

Uploading images and audio

  • Image and audio files should be uploaded in the Add/Update form before clicking submit. You will receive a message if the file is successfully uploaded.
  • Make a note of the file name and extension and enter it into the form so that it may be retrieved in the webpage. Be sure to use unique file names. If a file name already exists in the database, you will receive an error message when you attempt to upload it.
  • Once an image or audio file is uploaded, it may be used for multiple database entries. Just use the same file name.
  • Keep file sizes small for better performance. It is recommended that audio files be < 5MB and image files < 2MB. Images should not exceed a width of 300px. Remember, in general, the smaller the better!

Entering video embed links

The video embed template supports YouTube video embeds only. To enter the video link:

  • Go to the YouTube video page. Click the “Share” tab
  • Click the “Embed” tab.
  • Copy the “src” URL without the quotation marks.
  • Enter the URL into the Video Embed URL field of the Audio/Video table.

A note on apostrophes

When entering text that contains apostrophes into the form text fields, enter 2 single apostrophes instead of one to avoid an error. Use the following format:

The text 
Act now! Don't delay!
will produce an error. Instead enter 
Act now! Don''t delay!
